# Emogo - Emotion Tracker

## App Download
**Your app URI:** https://expo.dev/accounts/nickhung/projects/emogo-frontend/builds/44f753aa-a4fd-414f-91e4-c84671b4ab26

## Project Description
This is an emotion tracking application built with Expo Router. It allows users to:
- Record their current emotion (Happy, Calm, Sad, Angry)
- View history of recorded emotions
- Export data for analysis

## How to run

1. Install dependencies:

   ```bash
   npm install
   # or
   yarn
   ```

2. Start the dev server:

   ```bash
   npx expo start --tunnel
   ```

3. Open the app on a device or emulator using the Expo dev tools.
